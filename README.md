# Superstore Customer Analysis

## Objective
The objective of this project is to analyze customer purchasing behavior using sales data and identify key patterns such as customer segments, order frequency, and high-value customers.

---

## Tools Used
- SQL (for data analysis)
- Excel (for dataset)

---

## Dataset
The dataset contains information about customer orders, including:
- Order ID
- Customer ID
- Order Date
- Sales
- Profit
- Region and Category details

---

## Analysis Performed

1. Total number of records in dataset  
2. Date range of orders  
3. Unique customer count  
4. Top customers by revenue  
5. Customer segmentation based on order frequency  
6. Order frequency analysis  
7. VIP / high-value customers  
8. Yearly revenue trends  
9. Revenue by region and category  

---

## Key Insights

- 58% of customers fall under the loyal segment, indicating strong retention  
- A small group of customers contributes a significant portion of total revenue  
- Repeat purchasing behavior is observed across multiple customer segments  
- Certain regions and categories generate higher revenue compared to others
